网站开发日志

关于网站技术实现，以及迭代更新的记录。

2025-05-01GitHub

网站运行记录不定期绝赞更新中~ (∠・ω< )⌒★

网站基于 React 构建，并由 Next.js 框架驱动。在样式系统方面使用 Tailwind CSS v4 与 shadcn/ui 组件库实现响应式设计。

在 CI/CD 流程中通过 GitHub Actions 实现自动化代码质量测试与构建。其中使用 ESLint 进行代码静态分析与质量规范检查，使用 Prettier 进行代码格式化统一风格。并使用 Jest 进行 Unit Testing 测试。

就诊地图使用 Apache ECharts 进行数据可视化。图标资源使用 RemixIcon 图标库。

博客方案在内容处理方面使用 next-mdx-remote 加载 MDX 文件。结合 remark-GFM 扩展以支持脚注等 GitHub 风格的 Markdown 语法特性。

新网站的用户界面设计，我采用了偏向现代简约的风格，使用了圆角方形等清晰的视觉元素以提升可读性。在整体布局上有意留出了更充足的空白空间，最小化页面的冗余信息以避免分散读者的视觉重心，使得内容更为突出，帮助读者更好地集中注意力聚焦于关键内容，降低认知负担，使信息更易于阅读与吸收。
